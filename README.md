# Irish Spatial Data Infrastructure Metadata

_This repository is being developed to provide guidance and examples for best practice in metadata compilation for the Irish Spatial Data Exchange. The guidance notes are not intended to replace the ISDI Meteadata Standard documents, but to work in conjunction with it._
